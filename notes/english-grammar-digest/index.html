所有的语法只有一个目的：造句。

4 基本句式

4.1 简单句及谓语动词

  • 简单句(Simple Sentences)
什么 + 怎么样
  • 句子成分(Parts of Speech) 两个最基本的句子成分是：
什么？           怎么样？
主语(Subject) + 谓语(Predicate)
  • 几乎所有的英语句式都是
人或物   动作“干了什么”->(Verbs)
 主语 +  谓语

  • 有哪些“动作”(动词)？

    1. 可以独立完成的动作: 不及物动词(Intransitive verbs, 没有承受着的动词)

          Papa Cochon /sleeps/.

    主语 + （不及物）动词

    2. 有 一个 动作承受者: （单）及物动词 (Transitive verbs)

           Papa Dog /likes/ *you*.

     主语 + （单及物）动词 + 宾语

    3. 有 两个 动作承受者: 双及物动词 (Ditransitive verbs)

           Papa Duck /teaches/ *you* *English*.

     主语 + （双及物）动词 + "间接"宾语 + "直接"宾语

    4. 只有 一个 动作承受者（区别于 2）: 复杂及物动词 (Complex-Transitive verbs)

           Papa Chook /considers/ *you* ~smart~.

     主语 + （复杂及物）动词 + 宾语 + （宾语）补语

    5. 不是个“动作”: （连）系动词(Linking verbs)

           Papa Cat /is/ tall.         (Cat = tall)
     Papa Bull /is/ in the cage. (Bull = cage)
     Papa Sparrow /looks/ big.   (Sparrow = big)

                          ~表语~
     主语 + （系）动词 + （主语）补语

4.1.1 五种 “基本句型”： 几乎涵盖所有英语句型

1. 主语 +   （不及物）动词
2. 主语 +   （单及物）动词 + 宾语
3. 主语 +   （双及物）动词 + "间接"宾语 + "直接"宾语
4. 主语 + （复杂及物）动词 + 宾语 + （宾语）补语
5. 主语 +      （系）动词 + （主语）补语

4.1.2 小结

  • 谓语都有核心动词：谓语动词
什么？           怎么样？
主语(Subject) + 谓语(Predicate)
                ~谓语都有核心动词~
  • 谓语动词有五个类别：-> 五种基本句式
1. 无宾语（主动）
2. 一个宾语（主谓宾）
3. 两个宾语（间接 + 直接）
4. 一个宾语 + 补语
5. 赋予状态（主动表）

严格说：谓语 = 谓语动词 + 宾语

通常会把"谓语动词"直接称为 “谓语“ (主语 + 谓语动词 + 宾语 -> 主谓宾)

4.1.3 其他句子成分(Parts of Speech)

  1. 6). 定语(Attributive): 修饰主语或宾语

        ~The little white~ *rabbit* ate ~a large~ *carrot*.
    修饰主语 rabbit                修饰宾语 carrot

  2. 7). 状语(Adverbial): 修饰谓语动词

        The rabbit /ate/ ~quickly~
                     修饰 ate

  3. 8). 同谓语(Appositive): 再把主语或宾语说一遍

       Papa Toad, ~an ugly man~, wants to swallow a swan.
              和主语同等

  • 复杂句: Complex Sentences

5 词类

5.1 同一句型不同词类

句型相同，词类可以不同

     主语      谓语动词    宾语
  The rabbit    ate     a carrot.
      He        saw     someting over there.

5.2 十大词类

  1. 名词 Nouns
  2. 冠词 Articles
  3. 代词 Pronouns
  4. 形容词 Adjectives
  5. 数词 Numerals
  6. 副词 Adverbs
  7. 介词 Prepositions
  8. 叹词 Interjections
  9. 连词 Conjunctions
  10. 动词 Verbs

5.3 句子成分 vs 词类

句子成分和句子中的词类是两种完全不同的概念！ 除了谓语动词是动词，其他都可能包含不同词类。

5.4 谓语动词“三大本领”

  1. 表示动作的时间；(时)
  2. 表示动作的状态；(体)
  3. 表示动作的假设、情感等（语气，语法的语气）；(气)
   时       体        气
[T]ense  [A]spect  [M]ood

TAM (特么真难)

5.4.1 时态(难点)

即：时间 + 状态

  • 时间：现在，过去，将来，过去将来；
  • 状态：一般，完成，进行，完成进行；
时态的组合(4 * 4 = 16)

  现在              一般
  过去              完成
  将来              进行
过去将来           完成进行

5.4.2 动词语气(难点)

  • 虚拟语气：If I were a bird
  • 陈述语气：I ate a elephant and…
  • 祈使语气：/Catch/ this snake and throw to Trump

5.4.3 助动词

光凭借谓语动词还不能实现“三大本领”, 这时候就需要 助动词 了

5.5 谓语动词 vs 非谓语动词

动词除了充当句子中的谓语动词外，还有可能充当 1). 主语、2). 宾语、3). 宾语补语、4). 主语补语、 5). 定语、6). 状语、7). 同位语。=> 非谓语动词

非谓语动词几乎可以取代所有的从句，从而简化句子。非谓语动词不再具有表现 TAM 的功能。

5.5.1 非谓语动词的不同形式(难点)

  • 动词不定式

      I'm /going/ ~to eat~

  • 现在分词

  • 动名词

      The rabbit is /eating/.
  ~Eating~ everything, I'm survived.
  ~Eating~ doves is healthy.

  • 过去分词

      The rabbit has /eaten/.
  This is a ~half-eaten~ carrot.

6 核心：动词

英语语法的 核心 就是 动词

7 结语

首先要说明下，推荐你去看这个视频，不是说指望你能把英语语法学得多好。我自己也不会 语法，视频中讲的很多内容我看之前也是一窍不通。

推荐你看这个视频主要有两点吧，其一是觉得这个视频总结得很到位，把整个语法体系的 框架 脉络梳理得很清晰，把整个语法体系最简化，剩下的就是添砖加瓦的工作了； 其二 是我认为学习大抵就是个总结的过程，这也是为什么要求你写写笔记，因为写笔记的过程其 实也是个总结的过程。通过这个视频，你应该能认识到英语语法这么大的一个问题可以简化 到 “动词” 这个问题上，举一反三，其他学科知识点也类似，大体上的学习过程都是：学 + 总结 + 稍稍练习 + 总结 。

其次，要明白在高考这座“大山”面前，要学好一知识点或者说一门学科，并不难，要考好， 真的一点都不难（假设考好定义为 80 分以上，当然你可以有更大的野心），这是战略上的藐 视。考不好可以有一堆理由，但大都是借口，归结来说还是方法上的问题，战术不对。

其他心态什么的先不讲了，考好点，虽然没什么用，成就感还是有的。回到英语语法的问题 上，我自己的学习方式不一样，我基本没学过语法，或者说就是没学会过。我做对的大部题 都是凭借“语感”做的，语感这个东西不好说，举个例子，像我们说的土话、普通话也有语法， 汉语的语法甚至比英语难多了，文言文呢？但你从来没觉得说起话来语法有啥问题，“语感“ 差不多就是这种感觉。当然这里不是说语法没啥用，能学好就去学好吧，我想指出的是方 法可以有很多，你要自己去总结，这里仅作为抛砖引玉。

另外再啰嗦下，去享受学习带来的成就感，学习会伴随你漫长而又短暂的一生，养成好的总 结习惯或着说培养一个 现实主义思维 可能会让你的生活更轻松、更享受一点。稍微拉长 远点看，高中的时候目标很明确，该学啥学啥，大学呢想学啥学啥，出了社会被毒打的时候 则不得不学点啥。。。总的来说都是要去享受学习这个过程以及掌握之后能给你心理上或者 成绩上带来的某种成就感，当然对你来说可能考第一的时候是最爽的。
